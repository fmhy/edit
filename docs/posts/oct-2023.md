### Things Removed

- Removed RSLOAD, as they have uploaded a version of Malwarebytes bundled with
  the same malware that caused the recent FileCR controversy. We have emailed
  them; hopefully, they remove it soon so we can re-evaluate.

- Removed marin.moe from Anime Streaming as it is redirecting to an unrelated
  domain now.

- Removed AdGuard Extra and other adblock defenders as they are
  [redundant](https://ibb.co/xgrtbBZ) when using uBlock Origin.

- Removed SteamSkins as Steam dropped support for them in the latest update.

- Removed Saikou from Android Anime as their GitHub repository has been DMCA'd.

- Removed Uptobox from File Hosts as their data center was
  [raided by French police](https://torrentfreak.com/uptobox-goes-dark-following-police-raids-on-french-datacenters-230920/).
