# Stars Added ‚≠ê

- Re-starred [Copilot](https://fmhy.net/ai#official-model-sites) in Official Models and moved it to the #2 spot. It now has unlimited o4-mini and GPT-5 access.

- Starred [DeepL](https://fmhy.net/text-tools#translators) in Translators. Fast, accurate, and features flexible post-processing, making it ideal for production environments.

- Starred [UnionCrax](https://fmhy.net/gaming#download-games) in Game Downloading. Features fast hosts, pre-installs, and allows for requests.

- Starred [VORAPIS](https://fmhy.net/social-media-tools#youtube-customization) in YouTube Customization. Restores the classic layout and improves performance.

- Starred [WiiLink](https://fmhy.net/gaming-tools#wii-u-wii-homebrew) in Wii Homebrew. Restores functionality to Wii Channels that have been abandoned by Nintendo.

- Starred [AN1](https://fmhy.net/mobile#modded-apks) in Android APKs. Features a large library of games, quick updates, and single-click downloads.

- Starred [Namida](https://fmhy.net/mobile#android-audio-players) in Android Audio Players. Smooth, highly customizable, includes YouTube support, and features a nice UI.

- Starred [StartAllBack](https://fmhy.net/system-tools#system-tweaks) in System Tweaks. Restores Win10-style UI features in Win11 and improves overall responsiveness.

- Starred [Z.ai](https://fmhy.net/ai) in AI Chatbots. Fast, unlimited, and performs well; it's one of the better open-source reasoning models available.

- Starred [GAM.ONL](https://fmhy.net/gaming#browser-emulators) in Browser Emulators. Features a very nice UI and navigation system plus many consoles and games.

- Starred the [Vita Hacks Guide](https://fmhy.net/gaming-tools#playstation-homebrew) in PlayStation Homebrew. Well-respected, easy to follow, and highly recommended for anyone starting Vita modding.

- Starred [Subtitle Edit](https://fmhy.net/video#subtitle-tools) in Subtitle Tools. An easy-to-use, feature-rich, and well-maintained subtitle editor.
