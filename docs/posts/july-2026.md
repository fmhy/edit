# Stars Added ⭐

- Starred [ArchiveTune](https://fmhy.net/mobile#youtube-music) in Android YouTube Music Apps, has a nice UI and seems to work well for people.

- Starred [Too Good To Go](https://fmhy.net/misc#food) in Food section, good way to get cheap food, and prevent food waste, worked well for our members who tried it.

- Starred [Hardcover](https://fmhy.net/reading#tracking-database) in Reading Tracking / Databases, has a nice UI, lots of features, and ability to import from other platforms (like goodreads).

- Starred [Yasosu](https://fmhy.net/text-tools#pastebins) in Pastebins, has markdown support, allows editing posts, and has custom URLs.

- Starred [The Criterion Closet](https://fmhy.net/video#curated-recommendations) in Curated Recommendations, 3D Criterion walk-in closet recreation with filters.

- Starred [LibrePods](https://fmhy.net/linux-macos#linux-audio) in Linux audio, allows you to use AirPod features on Linux (and android rooted).

- Starred [SeaDex](https://fmhy.net/video#anime-torrenting) in Anime Torrenting, anime video quality guide, can be used in combination with things like Stremio + AIOStream.

- Starred [LACT](https://fmhy.net/linux-macos#linux-system) in Linux System, feature-rich GPU config + monitor, simplified overclocking.

- Starred [Unsloth](https://fmhy.net/ai#machine-learning) in Machine Learning, easy to use finetuning + AI suite, seems to be well liked.

- Starred [Zperiod](https://fmhy.net/educational#periodic-tables) in Periodic Tables, clean interactive periodic table w/ 3D atom models.

- [Bookracy](https://fmhy.net/reading#ebooks) is back and has been re-added & re-starred in Ebook section.

- Both [Soft98 + DownloadHa](https://fmhy.net/downloading#software-sites) are working again, they've been re-added & re-starred in the software section.
