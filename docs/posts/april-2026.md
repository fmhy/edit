# Stars Added ⭐

- Starred [LibreWolf](https://fmhy.net/privacy#browser-privacy) in Browser Privacy. Custom privacy-focused version of Firefox, easily accessible, has good configs, and is well liked.

- Starred [Tuta](https://fmhy.net/privacy#email-privacy) in Email Privacy. Encrypted email, gives 1GB free, preferred over things like Proton Mail by some. 

- Starred [Kettu](https://fmhy.net/mobile#social-media-apps) in Mobile Discord Clients. Modded client for both Android + iOS with plugin support, has been working well for those who have been testing it. 

- Starred [Nuvio](https://fmhy.net/mobile#android-streaming) in Android Streaming. Was already starred in iOS, but Android users seem to be finding it useful. Has plugin support (including stremio), a nice UI, and weekly updates.

- Starred [Qwen](https://fmhy.net/ai#official-model-sites) + [Ask Brave](https://fmhy.net/ai#specialized-chatbots) in AI section. Both are unlimited, and their models seem to be working well for people.

- Starred [The OPSEC Bible](https://fmhy.net/privacy#privacy-indexes) in Privacy Indexes. In-depth and well respected online anonymity guide.

- Starred [Is it really FOSS](https://fmhy.net/downloading#foss-sites) in FOSS section. Database that helps determine if something is truly FOSS or not.

- Starred [ScriptCat](https://fmhy.net/internet-tools#userscripts) in Userscript Managers. Open-source, MV3 compatible, allows background + scheduled scripts, considered by some to be the best current script manager.

- Starred [Readest](https://fmhy.net/reading#ebook-readers) in Ebook Readers. Customizable, supports many formats and platforms, gives free cloud storage w/ progress sync.

- Starred [Flash Museum](https://fmhy.net/gaming#multi-game-sites) in Browser Game Sites. Has a clean UI + one of the bigger libraries of oldschool flash games.

- Starred [PressReleased](https://fmhy.net/internet-tools#paywall-bypass) in Paywall Bypass. Based onBypass Paywalls Clean, requires no extension.

- Starred [Evade](https://fmhy.net/internet-tools#redirect-bypass) in Redirect Bypass. Fast, well maintained, and works on lots of sites.

- Starred [Wiktionary](https://fmhy.net/educational#dictionaries-thesaurus) in Dictionary section. Well respected dictionary with lots of active contributors.

- Starred [Mullvad + IVPN](https://fmhy.net/privacy#vpn) in VPN section. Both are confirmed [No-Logging](https://mullvad.net/en/blog/2023/4/20/mullvad-vpn-was-subject-to-a-search-warrant-customer-data-not-compromised/) / [2](https://www.ivpn.net/knowledgebase/privacy/how-do-we-react-when-requested-by-an-authority-for-information-relating-to-a-customer/), accept multiple payment types, and require no email.

- Re-starred [HolyUnblocker](https://fmhy.net/privacy#proxy-sites) in Proxy Sites. Working well again, has built-in adblock + Tor support.

# Things Removed

- Removed P-Stream as their owners have decided to [step back from the project](https://i.ibb.co/6RRB4YBj/image.png) / [2](https://i.imgur.com/xYIO9aK.png).

- Removed Tampermonkey as it's had various issues over the years with things like their privacy policy, going closed-source, updates causing data loss, etc. and there's better options anyways (ScriptCat, Violentmonkey, Firemonkey).

- Removed RTINGS as they've gone [fully paid](https://www.rtings.com/company/revamping-our-membership-program).

- Unstarred ReVanced as Morphe is now stable enough to replace it + has more active development.

- Unstarred Trakt as they're now [forcing users onto v3 design](https://redd.it/1rwgf8y) which is missing many features compared to previous version.
