# Stars Added ⭐

- Starred [r/CrackWatch](https://fmhy.net/gaming-tools#gaming-tools) in Gaming Tools, popular and useful piracy game release tracking group.

- Starred [HizSearch](https://fmhy.net/gaming#download-games) in Game Download, multi-site game download search engine, uses trusted sources, has a nice UI, and active development.

- Starred [NASA2Explore](https://fmhy.net/educational#astronomy) in Astronomy section, the official source of all NASA images, including full metadata.

- Starred [LocalTube Manager](https://fmhy.net/social-media-tools#youtube-tools) in YouTube Tools, extension that allows you to use YouTubes features without an account, including subscribing, playlists, and notifications, also allows you to export data to sync across devices.

- Starred [OpenCode](https://fmhy.net/developer-tools#coding-agents-extensions) in AI Coding Agents, has good free models + seems to work well for people.

- Starred [DOS Zone](https://fmhy.net/gaming#browser-emulators) in Browser Emulators, retro PC browser emulator that works well + has a big library.

- Starred [Shizuku Fork](https://fmhy.net/mobile#android-device) in Android Device, fixes multiple features, adds QoL updates, and is maintained well.

- Starred ⁠[SteaMidra](https://fmhy.net/gaming-tools#steam-epic) in Steam Tools, multi-tool that includes DLC unlockers, multiplayer fixes, and a manifest tool.

- Starred [ReadYou](https://fmhy.net/mobile#android-rss-readers) in Android RSS Readers, clean design, offline support, Google Reader + Fever API sync

- Starred [Gadgetbridge](https://fmhy.net/mobile#android-utilities) in Android Utilities, open-source gadget manager w/ support for 300+ devices and no account required.

- Starred [SmartTube](https://fmhy.net/video#android-tv-firestick) in Android TV, works on lots of TVs, has improved navigation, and is preferred over TizenTube by some.

- Starred [Lucas Chess + Chess Tempo](https://fmhy.net/educational#chess) in Educational Chess section. Feature-rich chess learning platforms both of which seem to be well liked and good for beginners.

- Starred [nakarte](https://fmhy.net/misc#maps) in Maps section, world map with multiple layers and street views all combined into one.

- Starred [OpenFront](https://fmhy.net/gaming#strategy) in Browser Strategy Games, open-source Conquest / War style game with active multiplayer community.

- Starred [setlist.fm](https://fmhy.net/audio#concerts-live-shows) in Concerts / Live section, in-depth database of setlists for almost every concert and tour in history.

- Starred [GiveWell](https://fmhy.net/misc#assistance-charity) in Charity / Donations section, charity research group, publishes their research process online, and charges no fees.

- Added [Spicy Lyrics](https://fmhy.net/audio#spotify-tools) next to Spicetify. Improved lyrics with very nice UI, not technically a star, but people seem to really like it so it felt worth mentioning.
