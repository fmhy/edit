- Added 360 Total Security to unsafe sites. If you install any of their apps, it will keep giving the user popups to install their "toolbox," or will install it without consent if you use clean, repair, or optimize options. # Stars Added ‚≠ê

- Starred [DDLBase](https://fmhy.net/video#download-sites) in Video Download. Has both 4K + 1080p content, uses only fast hosts, and has a huge library.

- Starred [Anime Realms](https://fmhy.net/video#anime-streaming) in Anime Streaming. Good sources, nice UI, auto-next, custom player, and multi-language support.

- Starred [Music Hunters](https://fmhy.net/audio#telegram-bots) in Telegram Audio Bots. Supports lots of sites, has good quality, and is easy to use.

- Starred [AstralGames](https://fmhy.net/gaming#download-games) in Game Downloading. Big library, fast hosts, pre-installs, and allows requests.

- Starred [Comix](https://fmhy.net/reading#manga) in Manga Sites. Big library, allows uploads, nice UI, and will be adding more languages soon.

- Starred [Monochrome](https://fmhy.net/audio#streaming-sites) in Audio Streaming. No sign-up needed, multi-API integrations, Last.fm integration, customizable, and a big library.

- Starred [Rive, Aether, Willow, and FlyX](https://fmhy.net/video) in Streaming. These have nice custom players, lots of features, fast streams, and impressive UIs.

- Starred [StreamSports99](https://fmhy.net/video#live-sports) in Live Sports. Has fast servers and a solid UI.

- Starred [Control Panel for Twitter](https://fmhy.net/social-media-tools#twitter-x-customization) in Twitter/X Customization. Adds a lot of extra features, lets you hide sections you don't use, improves UI, and much more.

- Starred [Rootz](https://fmhy.net/file-tools#file-hosts) in File Hosts. Good speed, 25GB per file, and lasts 15 days after the last download. 

- Starred [PigenAI](https://fmhy.net/ai#image-generation) in Image Gen. Has unlimited Imagen 4, Qwen, and Nano Banana.

- Starred [Nuvio](https://fmhy.net/mobile#ios-streaming) in iOS Streaming. This uses Stremio plugins and has fewer limits than the "Lite" version currently available on iOS.

- Starred [Flud](https://fmhy.net/mobile#android-torrenting) in Android Torrenting. Allows binding, is well-maintained, and ads can easily be blocked via DNS.

- Starred [Sleeping Guide](https://fmhy.net/misc#physical-health) in Physical Health. A sleep hygiene/science guide; seems to be well-received so far, curated by Dan.

- Starred [OpenMW](https://fmhy.net/gaming#remakes-recreations) in Remakes / Recreations. Complete ground-up reimplementation of Morrowind, [approved by Bethesda](https://wiki.openmw.org/index.php?title=Bethesda_Emails) themselves.

- Starred [Clone Hero](https://fmhy.net/gaming#remakes-recreations) in Remakes / Recreations. Feature-rich Guitar Hero clone, thousands of songs + playlists, no lag, works with all guitar controllers, and supports MIDI drum kits.

- Starred [Gidd](https://fmhy.net/gaming#party-multiplayer) in Multiplayer Browser Games. Has quality versions of things like Monopoly, GeoGuessr, card games, etc. 

- Starred [QEMU](https://fmhy.net/system-tools#virtual-machines) in Virtual Machines. Performs well on non-Windows systems; best used with Virt-Manager or Vagrantup.

- Starred [Restic](https://fmhy.net/file-tools#file-backup) in File Backup. Easy to use, multi-platform, and performed well in benchmarks.

- Starred [Gemini CLI](https://fmhy.net/ai#coding-ais) in Coding AIs. This seems to have some of the best limits as of now.

# Things Removed

- Removed CrocDB as they've been [hit by DMCA](https://i.ibb.co/RpDvJFgf/image.png) / [2](https://i.imgur.com/iSV0MHq.png). 

- Removed the IcebergCharts website as they've been [hacked a few times now](https://i.ibb.co/XxMMwqDk/image.png) / [2](https://i.imgur.com/MkvElvr.png), and can't figure out why it's happening. We may re-add it in the future if they figure it out. Note that we'll keep the subreddit in its place for now.

- Removed CanvasBlocker as it's Firefox-only, and FF has this built-in now; it can also apparently make fingerprinting worse in some cases and isn't maintained well anymore.

- Removed FTV and WAC from Live Sports as both are now gone.

- Removed Foxified as it's suddenly [filling results with ads](https://i.ibb.co/4RzPsQ8Z/image.png) / [2](https://i.imgur.com/AjBMdfX.png). 

- Unstarred OnTheSpot as it will be losing Spotify support soon.

- Unstarred Windsurf in Coding AIs as its limits and free models aren't as good anymore.
