# Wiki Updates

- Our sites default search is now **[Exact Match](https://i.ibb.co/rKNb8CGh/34t34g34.png) / [2](https://i.imgur.com/6XUs4fs.png)** which is a lot more accurate. You can toggle between both exact and fuzzy. TY to Zenith for doing this.

# Stars Added ⭐

- Starred [Hydra](https://fmhy.net/gaming-tools#game-launchers) in Game Launchers, has plugin-based downloads, torrents, RD + TorBox, good customization, achievements, friends list, playtime monitor, cloud saves, etc. ⁠

- Starred [SpotiFLAC-Mobile](https://fmhy.net/mobile#android-audio) in Android Audio, open-source download app with support for multiple sites.

- Starred both [ElyPrismLauncher + Freesm Launcher](https://fmhy.net/gaming-tools#launchers) in MC Launchers, both are feature-rich forks of Prism that stood out to people more than other forks we tried.

- Starred [Gemini](https://fmhy.net/ai#official-model-sites) in AI Chatbots, Gemini 3 Flash works well now and is ranked highly on LMArena.

- Starred [Morphe](https://fmhy.net/mobile#android-youtube-apps) in Android YouTube + Android YouTube Music, ad-free YouTube patcher / ReVanced alternative.

- Starred [XDMovies + The One Piece](https://fmhy.net/video#download-sites) in Video Downloading, both have big libraries + fast hosts. 

- Starred [mpvEx](https://fmhy.net/mobile#android-video-players) in Android Video Players, feature-rich mpv fork, improved UI, well-maintained.

- Starred [Virgil Game Search](https://fmhy.net/gaming#download-games) in Game Download, custom multi-site game download search engine, uses only trusted sources.

- Starred [Dan Store](https://fmhy.net/downloading#freeware-sites) in Freeware Sites, web-based Microsoft Store explorer, bypasses need for Microsoft Store app.

- Starred [KoReader](https://fmhy.net/reading#ebook-readers) in Ebook Readers, open-source, highly customizable, plugin support, very good for e-readers.

- Starred [Nothing To Watch](https://fmhy.net/video#recommendation-tools) in Movie Recommendation Tools, interactive discovery visualization, fun / unique way to find something to watch.

- Starred [Ekvis](https://fmhy.net/educational#geography-sociology) in Geography, well-maintained and useful geography learning site, covers 220+ countries, flags, territories, etc.

- Starred [Slope Plus](https://fmhy.net/gaming#browser-games) in Browser Games, updated version of Slope, has extra features, game modes, and a active custom map community.

- Starred [GeminiGen AI](https://fmhy.net/ai#video-generation) in Image + Video Gen, gives unlimited Nano Banana Pro images, and unlimited Sora 2, Veo 3.1 and Grok videos.

- Starred [Perchance](https://fmhy.net/ai#image-generation) Image Gen, unlimited, no sign-up required, has improved a lot quality wise.

- Starred [Flow-Fi](https://fmhy.net/audio#lofi-radio) in Low-Fi Radio, has lots of options for stations and video backgrounds, higher quality than most lo-fi sites we're sent.

- Starred [StreetComplete](https://fmhy.net/misc#navigation-transport) in Navigation section, help fill in missing OpenStreetMap data by completing quests.

- Starred [Flash Arch, Kongregate & Crazy Games](https://fmhy.net/gaming#multi-game-sites) in Browser Games. After going through this section, these 3 seemed to stand out the most to people, but if we missed any other good ones feel free to let us know. We did also consider AddictingGames + Poki, but we were too unsure on both.
